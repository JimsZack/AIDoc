# 历年真题与解析

本文件夹收录系统架构设计师综合知识科目历年高频真题与详细解析，方便对照复习。

## 目录结构

```
历年真题与解析/
├── README.md                  ← 本文件（总览与使用说明）
└── 架构设计模块/
    ├── 01-架构风格真题.md
    ├── 02-质量属性真题.md
    ├── 03-设计模式真题.md
    ├── 04-架构评估真题.md
    └── 05-4+1视图与DSSA真题.md
```

## 使用方法

1. **先做题**：每道题先不看答案，自己判断选项
2. **看解析**：核对答案后重点看解析中的关键词和易错分析
3. **回顾 README 主文档**：将做错的题对应回 README 中的知识点表格，强化记忆
4. **标记错题**：在自己的复习笔记中标注反复出错的题型

- 2015–2025 年系统架构设计师综合知识真题

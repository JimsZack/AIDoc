## 使用方法

1. **先做题**：每道题先不看答案，自己判断选项
2. **看解析**：核对答案后重点看解析中的关键词和易错分析
3. **回顾 README 主文档**：将做错的题对应回 README 中的知识点表格，强化记忆
4. **标记错题**：在自己的复习笔记中标注反复出错的题型
5. **按模块突击**：考前按模块逐个过关，每个模块至少刷 2 遍

## 复习建议

- 架构设计模块：**必须拿下**，是分值最高的模块
- 软件工程与项目管理：**规则性强**，背诵口诀即可
- 数据库与操作系统：**公式计算**，重点练习 PERT、范式判断
- 安全与知识产权：**记忆型**，适合考前冲刺
